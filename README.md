# 🚀 AI Resume Analyzer

An AI-powered web app that extracts resume details and generates a score.

## 🔥 Features
- Resume parsing (PDF)
- Skill extraction
- Resume score (ATS-style)
- Dark/Light UI

## 🛠️ Tech Stack
- Python
- Flask
- Tailwind CSS

## 🚀 Run
pip install -r requirements.txt  
python app.py  

## 🌐 Usage
Open: http://127.0.0.1:5000/

## 📊 Score Logic
score = len(skills) * 10
